# AgentOS Skills 使用说明

## 什么是 Skills？

Skills 是 AgentOS 的技能扩展系统。每个 Skill 赋予 AI 助理一项具体能力（如查天气、翻译文本）。对话时 AI 会自动判断是否需要调用某个 Skill，你无需手动触发。

## 查看已安装的 Skills

打开 Skills 面板（侧边栏或设置中的 Skills 入口），默认显示 **Installed** 标签页，列出当前已安装的所有技能。

每个技能卡片包含：

- **名称 + 版本号**
- **功能描述** — 这个技能能做什么
- **审核标识** — Official（官方）/ Reviewed（已审核）/ Unreviewed（未审核）
- **环境标签** — Cloud（云端执行）/ Desktop（桌面端执行）
- **作者信息**

## 浏览 Skill 库

切换到 **Library** 标签页，可以浏览所有可用技能。每个技能会显示当前安装状态：

- **Install** 按钮 — 点击安装该技能
- **Installed** 标识 — 已安装

## 安装与卸载

| 操作 | 方法 |
|------|------|
| 安装 | Library 标签页，点击技能卡片上的 **Install** |
| 卸载 | Installed 标签页，点击技能卡片上的 **Uninstall** |
| 刷新列表 | 点击右上角 **Refresh** 按钮 |

注册时系统会自动安装默认技能（天气查询、文本翻译），可按需卸载。

## 在对话中使用

Skills 是**自动触发**的，不需要特殊指令。举例：

| 你说 | AI 自动调用 | 效果 |
|------|------------|------|
| "北京今天天气怎么样" | weather 技能 | 返回实时天气数据 |
| "把这段话翻译成英文" | translate 技能 | 调用翻译并返回结果 |

对话过程中，如果 AI 正在调用某个 Skill，界面底部会短暂显示技能名称和执行状态。

**重要**：只有已安装的技能才会在对话中被调用。卸载某个技能后，AI 将不再具备该能力。

## 当前可用技能

| 技能名 | 功能 | 环境 |
|--------|------|------|
| weather | 查询全球城市实时天气（支持中英文城市名） | Cloud |
| translate | 多语言文本互译，自动检测源语言 | Cloud |

## 不同模式下的行为

| 模式 | 可管理 | 说明 |
|------|:------:|------|
| 内置助理 (builtin) | Yes | 完整的 Installed + Library 双标签，支持安装/卸载 |
| BYOK | Yes | 同内置助理 |
| OpenClaw 托管 | View only | 仅显示已安装列表，技能由平台管理 |
| OpenClaw 自建 | View only | 显示 Agent 上的技能列表，管理权在你自己的 OpenClaw |
| CoPaw | View only | 显示 CoPaw Agent 的技能列表 |

## 注意事项

- 未登录用户可以使用所有默认公开技能，但无法管理安装状态
- 技能列表有本地缓存，如遇数据不一致请点击 Refresh
- 私有技能（visibility: private）仅对其 owner 可见
